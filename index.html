BBCode Introduction

BBCode is a simple markup language that is used by bulletin boards, blogs, forums, and other similar sites. Below are some tips on how to use it.

Text Styling

Text can be styled in BBCode using simple tags similar to those used in HTML. Below is a list of some of the more commonly used tags.

  • Bold text is applied using the format: [b]{text}[/b]
  • Italics uses: [i]{text}[/i]
  • Underline: [u]{text}[/u]
  • Strikethrough: [s]{text}[/s]
Alignment

Text can be aligned left, center, and right. There is not a tag for justifying content that I'm aware of.

  • Left: [left]{text}[/left]
  • Center: [center]{text}[/center]
  • Right: [right]{text}[/right]
Links

Simple links are handled using the syntax [url]{url}[/url].

Links can be named as well: [url={url}]{url}[/url].

Images and Video

Guess how images are displayed in BBCode!! It's going to look pretty familiar: [img]{url}[/img].

YouTube videos work too: [youtube]{id}[/youtube].

Tables

Like HTML, BBCode can generate tables. [table]{rows}[/table] builds a row-based table. Cells are filled using [th]{content}[/th] for table headers and [td]{content}[/td] for regular cells.

Lists

BBCode can do lists too. Use [ol]{items}[/ol] for ordered lists and [ul]{items}[/ul] for unordered lists. Items are entered with [li]{item}[/li].

Quotes and Code

Quotes are handled with the [quote]{text}[/quote] syntax.

Code uses [code]{text}[/code].
